KEY FEATURES SUMMARY FOR PRESENTATION
================================================================================

PATIENT ROLE - 3 Main Features:
  1. Disease Predictions (3 types: Diabetes, Heart, Kidney)
  2. Prediction History (View and track all predictions)
  3. Profile Management (Update personal and medical info)

DOCTOR ROLE - 3 Main Features:
  1. Patient Management (View and manage patient list)
  2. Prediction Reviews (Review and analyze predictions)
  3. High-Risk Alerts (Receive and act on critical alerts)

ADMIN ROLE - 3 Main Features:
  1. User Management (Manage all users in system)
  2. System Analytics (View statistics and reports)
  3. System Configuration (Configure system settings)
